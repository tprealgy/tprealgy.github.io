JavaScript functions

How do they work?

JavaScript functions are blocks of code that can be defined and then called later. They can take in parameters and return values. Functions can be declared in two ways: as a function declaration or as a function expression. Function declarations are hoisted, meaning they can be called before they are declared. Function expressions are not hoisted, so they must be declared before they are called.
