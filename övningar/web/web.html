LAB_0: Media queries

På webbsidan hittar du endast en nav med en enkel text som logotyp och ett par "länkar". Dolt finns även en bild för en "hamburger"-meny. Den syns inte på sidan, men du kan se den i koden

Att göra:

Använd koden och fyll i @media längst ned i style-taggen. Ditt mål är att ta bort länkarna när sidan är mindre än 600px bred och istället visa bilden. Du behöver endast skriva kod innanför @media { }

LAB_1:

Sidan du ser är byggd för att visa alla rutor på ett särskilt sätt baserat på CSS grid. Koden är dock inte färdig. Med korrekt kod så kommer sidan att få ett unikt useende där alla rutor placeras ut automatiskt

Att göra:

Fyll i body { } så att det blir ett grid-element och sätt kolumner i grid till 12 st där alla har bredden 1fr. Du behöver endast skriva kod i body { }-selektorn
