För den här övningen så förutsätts att du har både tillägget och din .scss-fil redo att använda. Om det inte stämmer eller om du är tveksam bör du vid det här laget be om hjälp.

  1. Ladda ner SCSS-filen
  2. Läs igenom filen - Se vad för CSS som finns i den.
  3. Starta "Watch Sass" - Din CSS genereras nu automatiskt.
  4. Läs igenom den genererade CSS-filen och jämför skilnaden mellan de två filerna
  5. Länka din nya-genererade CSS-fil till denna exempel-fil och se att det funkar.
